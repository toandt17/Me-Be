== Features ==

* Send new orders from wooccommerce to Facebook messenges
* Send new orders from wooccommerce to telegram id
* Messages quickly
* Convenient to use

=== From within WordPress ===

1. Visit 'Plugins > Add New'
2. Search for 'New order Notification for Telegram and Facebook'
3. Activate New order Notification for Telegram and Facebook from your Plugins page.

=== Manually ===

1. Upload the 'New order Notification for Telegram and Facebook' folder to the '/wp-content/plugins/' directory
2. Activate the New order Notification for Telegram and Facebook display plugin through the 'Plugins' menu in WordPress
3. Go to "after activation" below.

== Installation ==
1. Visit \Plugins > Add New
2. Search for \New order Notification for Telegram and Facebook\
3. Activate New order Notification for Telegram and Facebook from your Plugins page.
